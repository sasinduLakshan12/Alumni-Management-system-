# Alumni Management System

## Overview

The Alumni Management System is a collaborative web-based application developed to streamline alumni record management and strengthen communication between graduates and the institution. The system provides secure authentication, profile management, alumni tracking, and administrative functionalities through an intuitive user interface.

---

## Project Type

This is a **group project** developed collaboratively by a team of undergraduate students as part of academic coursework.

Each team member contributed to different parts of the system including frontend development, backend logic, database design, testing, and documentation.

---

## Key Features

### Alumni Module

* User Registration and Login
* Secure Authentication & Session Management
* Profile Management
* View Alumni Information
* Access Announcements and Updates
* Responsive Design

### Admin Module

* Admin Dashboard
* Alumni Record Management
* Add, Update, and Delete Alumni Data
* Search and Filter Alumni Information
* Announcement and Event Management
* Database Administration
